# Paisley Highland Games
This project involved the design and development of a proof-of-concept web application for Paisley Highland Games, completed collaboratively with another developer. The work included producing a specification document, designing the database schema, and implementing a functional website based on a defined project brief. Version control and containerisation tools were used to support collaborative development and deployment.
This project recieved an A grade.

## Brief
Develop a website for Paisley Highland Games that provides contextual information about the proposed competitions and enables groups or individuals to apply to participate in events.

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Frameworks & Libraries: React, Bootstrap
- Backend: Node.js with Express
- Database: MongoDB Atlas
- Authentication: Auth0
- Collaboration tools: Git, GitHub, Docker
- Design tools: Figma
- Deployment: Render


## Key Features
- CRUD Functionality with different views and features for admin, participants/authenticated users and guests
- Dynamic content rendering using React
- Authenticated and authorisation via Auth0
- Database driven content dynamically retrieved from MongoDB
- Participant registration
- Administrators can execute db queries via application interface: create, update and delete
- Image upload
